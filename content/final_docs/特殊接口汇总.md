## 特殊接口

![ChatGPT Image 2026年6月23日 14_55_03](D:\work\codex\final_docs\特殊接口汇总.assets\ChatGPT Image 2026年6月23日 14_55_03.png)

| 能力                 | 服务端接口                                                | 调用位置                                           | 作用                                                         |
| -------------------- | --------------------------------------------------------- | -------------------------------------------------- | ------------------------------------------------------------ |
| 常规模型采样         | responses                                                 | client.rs (line 147)                               | 普通 run_turn 请求模型，基线接口                             |
| 上下文压缩           | responses/compact                                         | client.rs (line 446) / compact.rs (line 32)        | 旧 remote compact，服务端直接返回 replacement history        |
| 记忆总结             | memories/trace_summarize                                  | client.rs (line 571) / memories.rs (line 32)       | 把 raw memory traces 总结成 memory_summary，注：当前生产流程没使用这个接口。 |
| 模型列表             | models                                                    | models.rs (line 31) / models_endpoint.rs (line 81) | 拉取远端 model catalog，带 client_version 和 ETag            |
| Web 搜索             | alpha/search                                              | search.rs (line 31) / tool.rs (line 73)            | web-search 扩展把搜索请求发给服务端，返回 encrypted output   |
| 图片生成/编辑        | images/generations, images/edits                          | images.rs (line 33) / backend.rs (line 40)         | image-generation 扩展调用图片生成或编辑接口                  |
| 文件上传             | files, files/{id}/uploaded，中间还有预签名 upload URL PUT | files.rs (line 97) / mcp_openai_file.rs (line 100) | 把本地文件上传成后端文件，返回 file_id、download_url、sediment://... |
| Realtime WebRTC 建连 | realtime/calls                                            | client.rs (line 542) / realtime_call.rs (line 60)  | 先用 HTTP 创建实时语音/WebRTC call，拿 SDP 和 call_id        |
| Realtime WebSocket   | /v1/realtime 或 sideband call_id WS                       | methods.rs (line 560)                              | 实时音频帧、session.update、response.create、实时转写/输出事件 |
